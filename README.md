# Mastery Curriculum Repository

This repository contains a 5-year comprehensive curriculum for mastering Oratory, Acting, and Debate, specifically tailored for a Software Engineer.

## Structure

- **Curriculum_Overview.md**: The master plan and philosophy.
- **Master_Reading_Log.md**: The complete bibliography.
- **Practice_Log_Template.md**: Daily tracking sheet.
- **Year_[1-5]/**: Detailed monthly and daily breakdowns.

## Getting Started

1. Read `Curriculum_Overview.md` to understand the philosophy.
2. Open `Year_1/Year_Strategy.md` to see the roadmap for the first year.
3. Begin with `Year_1/Month_01/Routine.md`.

## Core Philosophy: The 80/20 Rule
80% of your time should be spent on **Output** (Drills, Speaking, Writing). Only 20% on Input (Reading Theory).
